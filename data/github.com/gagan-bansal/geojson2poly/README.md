# geojson2poly
Converts GeoJOSN polygons to openstreetmap [(osm) poly format](http://wiki.openstreetmap.org/wiki/Osmosis/Polygon_Filter_File_Format) file.
## installation

```
npm install geojson2poly -g
```

## usage
```
geojson2poly <geojson-file> <output-poly-file>
```
